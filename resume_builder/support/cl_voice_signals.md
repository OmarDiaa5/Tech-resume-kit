# Cover Letter Voice Signals — Schema Reference

> **Runtime data is appended to `output/CL_VOICE_SIGNALS.md` (per-user, not committed).**
> This file defines the format only. Do not add personal data here.

## Format

Each entry block:

```
## [Company] ([date]) — [session path]
- "[verbatim user phrase]" — [signal note]
- Tone shift: [before] → [after]
```

Scope rules:
- Store wording signals only (verbatim or near-verbatim user language).
- Do not store structural or formatting signals (those go in resume_edit_signals.md).
- Newest entries at top.
